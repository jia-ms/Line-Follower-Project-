# Line-Follower-Project-
A Line Follower Robot using Arduino, L293D motor driver, and 4 IR sensors detects a black line on a white surface. The IR sensors send signals to the Arduino, which controls two DC motors via the L293D to move forward or turn. It demonstrates sensor interfacing, motor control, and basic embedded automation.

REQUIREMENTS

Required Library - AFMotor Library
Library Name: Adafruit Motor Shield Library (AFMotor)
Install from: Arduino IDE → Sketch → Include Library → Manage Libraries
Search: Adafruit Motor Shield Library
Install (Version 1 library, not V2)

Hardware Requirements

Arduino UNO/Nano
L293D Motor Driver Shield (Adafruit V1 type)
2 DC Motors
4 IR Sensors (Digital Output type)
Robot chassis + wheels
Battery (7V–12V)
Jumper wires
